Preguntas

¿Cuál es la diferencia entre padding y margin?

Padding es el espacio dentro de la caja. Margin es el espacio afuera de la caja.

¿Por qué se recomienda usar box-sizing: border-box?

Para que el padding y el border no hagan que la caja se agrande más. Los cálculos son más fáciles.

¿Qué significa diseñar con enfoque mobile-first?

Primero hacemos el CSS para móviles pequeños, y luego agregamos cambios para pantallas más grandes.

¿En qué situaciones preferirías rem sobre px?

Rem es mejor para tamaños de texto y espacios generales. Px es mejor para bordes y sombras pequeñas.
