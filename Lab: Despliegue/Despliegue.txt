3
Investiga los tipos de DNS y explícalos
Las DNS son las que convierten los nombres de dominio en direcciones IPS
A = relaciona el dominio con una direccion IPV4 
AAAA = La relaciona con una IPV6
CNAME = hace que un dominio apunte hacia otro 
MX = define que servidor recibe correos
TXT = Guarda texto informativo
NS = Name server 
PTR = REaliza resoluciones inversas
SRV = Define servicios especificos y puertos

¿Qué significa para un dominio cambiar el registro NS a diferencia de los demás?
Significa cambiar completamente quien administra ell DNS
